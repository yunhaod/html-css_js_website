Overview

This project aims to improve home security by notifying homeowners of any intrusion detected by a motion sensor. Upon connection and subscription with the buttons in the app, messages will appear below, indicating whether there appears to be a threat to the home.

More details about the project can be found in this GitHub Repository

Outcome

The app succeeds in receiving the messages from the sensor client upon any updates.

Introduction

This project utilizes MQTT(Message Queuing Telemetry Transport) for transmitting the messages between the clients. MQTT is a messaging protocol for IoT, which is ideal for establishing connection between remote devices. The MQTT broker, in particular, is responsible for relaying the messages for each topic within the network.

Local MQTT Broker Output

Iterations

The fist few iterations of the project were testing functionality of the system on a local MQTT broker that ran from my computer. After validating the MQTT system, the broker was hosted over the internet. A system of authentication was added to improve the security of the overall system, allowing only authorized users.

The messages appeared after subscribing, shown in the output console for the initial app design. In order to prevent API misuse, most buttons are hidden until the MQTT Manager is connected to the MQTT broker in the final version of app.

Hardware

An ESP32 was chosen to process the sensor data and for connecting to MQTT broker. The microcontroller determines whether there was an intruder through the use of a hall effect sensor.

A magnet is attached at the door, and when the door swings open, it will pick up on this event and notify the homeowner.

The warning will remain until the app publishes a message of acknowledgement.

To toggle the warning, a message of acknowledgment has to be sent from the app.

Takeaways

Through this project, I was able to learn more about IoT protocols, authentication systems, and backend integration in swift. I gained comprehensive insights on the fundamentals of MQTT and integrating it into real world applications.
